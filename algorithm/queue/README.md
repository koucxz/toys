# 队列

受限的线性结构，先进先出（First In First Out），不可插入和删除中间元素

- 入队，在队尾插入新元素
- 出队，删除队头的元素

## 优先队列
优先队列，入队时（也可以时出队时）需要考虑数据的优先级
先处理高优先级元素，同样优先级先进先出
